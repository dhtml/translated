A [Flarum](http://flarum.org) extension. DHTML Translated

git tag v0.0.0 && git push origin v0.0.0

## Installation

Install with composer:

```sh
composer require dhtml/translated:"*"
```

## Updating

```sh
composer update dhtml/translated:"*"
php flarum migrate
php flarum cache:clear
```
